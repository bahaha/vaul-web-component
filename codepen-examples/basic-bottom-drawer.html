Basic Bottom Drawer

Simple drawer implementation with Vaul Web Component

Welcome to the Drawer

This is a basic bottom drawer implementation using Vaul Web Component. You can drag the handle to dismiss it or click outside to close.

Features:

  • • Smooth slide-up animation
  • • Drag to dismiss functionality
  • • Click outside to close
  • • Keyboard accessible (ESC key)
  • • Focus trap for accessibility

Built with Vaul Web Component and Tailwind CSS
